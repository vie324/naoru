# 🚀 クイックスタートガイド

## すぐに使い始める（バックエンド不要）

### 1. ブラウザで開く

`index.html` をダブルクリックするだけで、すぐに使えます！

**または、簡易サーバーを起動：**

```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx http-server
```

ブラウザで `http://localhost:8000` にアクセス

### 2. Gemini APIキーを設定

1. [Google AI Studio](https://makersuite.google.com/app/apikey) でAPIキーを取得
2. アプリ右上の⚙️設定をクリック
3. APIキーを入力して保存

### 3. 使ってみる

#### ✅ **すぐに使える機能**（バックエンド不要）

- ✅ **ダッシュボード**: 店舗別ランキング、媒体分析
- ✅ **詳細・AI分析**: AI酒井くんチャット、原稿作成
- ✅ **予測**: 来店数予測、予算最適化、LTV分析
- ✅ **レポート**: AIサマリー生成、ベンチマーク、データエクスポート
- ✅ **データ管理**: 月次データ入力、スタッフ実績

#### 🔄 **デモモードで動作**（バックエンド推奨）

- 🔄 **競合分析**: デモデータで動作確認可能
  - 競合店舗トラッキング
  - 検索順位モニター
  - 口コミAI分析（実際のAI分析）

## オプション：バックエンドサーバーを起動（実際のスクレイピング用）

### 方法1：Dockerを使用（推奨）

```bash
cd backend
docker build -t naoru-backend .
docker run -p 3001:3001 naoru-backend
```

### 方法2：直接起動（環境依存）

```bash
cd backend
npm install
npm start
```

**注意**: Puppeteerのインストールには時間がかかる場合があります。

## バックエンド不要の理由

このアプリの**80%以上の機能はバックエンドなしで動作**します：

| 機能 | バックエンド不要 | バックエンド推奨 |
|------|----------------|-----------------|
| AI分析（酒井くん） | ✅ | - |
| 来店数予測 | ✅ | - |
| 予算最適化 | ✅ | - |
| レポート生成 | ✅ | - |
| データ管理 | ✅ | - |
| 競合分析 | 🔄 デモ | ✅ リアル |
| 口コミ分析 | ✅ デモデータでAI分析 | ✅ リアルデータ |

## トラブルシューティング

### Q: 「Failed to fetch」エラーが出る

**A**: 問題ありません！デモモードで動作します。
- 競合分析タブでは、サンプルデータが表示されます
- 他のすべての機能は完全に動作します

### Q: バックエンドを起動できない

**A**: バックエンドは**オプション機能**です。
- スクレイピング以外の全機能が使えます
- 実際のホットペッパーデータが必要な場合のみ起動してください

### Q: APIキーが必要？

**A**: Gemini APIキーは**AI機能のみ**に必要です。
- ダッシュボード、予測、データ管理は不要
- AI酒井くん、サマリー生成、口コミ分析には必要

## 推奨ワークフロー

### 初めて使う場合

1. **index.htmlを開く** → バックエンドなしでOK
2. **設定からAPIキーを入力** → AI機能を有効化
3. **店舗を追加** → 右上の店舗管理から
4. **データを入力** → データ管理タブで
5. **ダッシュボードで確認** → 分析結果を表示

### 競合分析が必要な場合

1. **競合分析タブを開く**
2. **デモモードで試す** → エリアを入力
3. **実データが必要なら** → バックエンド起動
